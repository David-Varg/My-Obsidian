> [!abstract]+ Resumen
> **Idea Principal**: Una variable es un nombre simbólico vinculado a una ubicación física en la memoria (RAM) que almacena un valor, mientras que el tipo de dato define la naturaleza, el tamaño y las operaciones permitidas sobre dicho valor.
> **Contexto**: Para un Ing. de Software, entender esto es crítico para optimizar el uso de recursos, evitar desbordamientos de memoria y garantizar la integridad de la lógica de negocio.

## 🎯 **Concepto Clave**
**Definición**: Las **Variables** son abstracciones que permiten al programador manipular datos sin gestionar manualmente direcciones hexadecimales de memoria. El **Tipo de Dato** actúa como un contrato que le indica al compilador/intérprete cuántos bytes reservar (ej. 4 bytes para un entero) y cómo interpretar esos bits (ej. complemento a dos para signos).

> [!tip] Intuición:
> Imagina una variable como una caja con una etiqueta. El "Tipo de Dato" es la forma de la caja: si es redonda, solo acepta pelotas; si es cuadrada, solo acepta libros. No puedes meter un libro en el hueco de una pelota.
