|               **Estructura**               | **Qué es realmente**         |
| :----------------------------------------: | ---------------------------- |

| [[06. Listas Enlazadas\|Listas Enlazadas]] | Nodos con punteros           |

Una computadora solo entiende cosas muy simples:
- Números
- Direcciones de memoria
- Bytes

Pero nosotros usamos cosas más complejas como:
- Listas
- Objetos
- Árboles 
- Grafos
- Strings
- Matrices

Entonces aparece la pregunta clave:
>¿Cómo representa la computadora esas estructuras complejas usando solo memoria?

La respuesta casi siempre es:
- **Usando direcciones de memoria (punteros/referencias).**

---
### Representación de Arrays
Hagamos un ejemplo en Python:
```python

a = [1,2,3]
```

Si hablamos conceptualmente, pasaría algo como esto:
```

STACK
a → 0xA100

HEAP
0xA100 → [1,2,3]
```

Pero eso no es realmente así por dentro, porque la lista misma contiene más estructura. En realidad se parece más a esto:
```python

a = [1,"hola",[5,6]]
```

Memoria:
```

STACK
a → 0xA100

HEAP
0xA100 = lista (Array)
    |
    |---0xB200 = 1
    |---0xC500 = "hola"
    |---0xD700 = lista (Array)
            |
            |---0xD80F = 5
            |---0xD30L = 6
```

Y luego claro, los datos se convierten en [[02. Binario y Lógica|Binario]].
Es literalmente un pequeño [[09. Grafos|Grafo]] de referencias.

---
### Representación de Objetos
Sigamos con Python:
```python

persona = {
    "nombre": "Ana"
    "edad": 22
}
```

En Memoria seria algo así:
```

STACK
persona → 0xA300

HEAP
0xA300 = Obj/Dict
    |
    |---0xB400 = "nombre" → 0xC300
    |---0xB500 = "edad" → 0xC400

0xC300 = "Ana"
0xC400 = 22
```

Los objetos son básicamente:
> **Tablas que apuntan a valores**

Eso es todo.

---
### Representación de Estructuras más complejas

Ahora viene lo que conecta con estructuras de datos reales.

Ejemplo: **Linked List**
```js

class Nodo {
    constructor(valor){
        this.valor = valor
        this.siguiente = null
    }
}
```

Memoria:
```

0xA100 → valor: 5
    next → 0xA200
 
0xA200 → valor: 7
    next → 0xA300

0xA300 → valor: 9
    next → null
```

Visualmente:
```

5 → 7 → 9
```

Pero realmente es esto:
```

5(0xA100) → 7(0xA200) → 9(0xA300)
```

Esto solo funciona gracias a [[04. Punteros|punteros]].

### Árboles
```js

const arbol = {
    valor: 10,
    izquierdo: {
        valor: 5,
        izquierdo: {
            valor: 2,
            izquierdo: null,
            derecho: null
        },
        derecho: {
            valor: 7,
            izquierdo: null,
            derecho: null
        }
    },
    derecho: {
        valor: 15,
        izquierdo: null,
        derecho: {
            valor: 20,
            izquierdo: null,
            derecho: null
        }
    }
}
```

En Memoria:
```

    10
    / \
   /   \
  5     15
 / \      \
2   7     20
```

Representación:
```

0xA100 → valor: 10
    izquierda: 0xB100
    derecha: 0xB200

0xB100 → valor: 5
    izquierda: 0xC100
    derecha: 0xC200

0xB200 → valor: 15
    izquierda: null
    derecho: 0xC600

0xC100 → valor: 2
    izquierda: null
    derecha: null

0xC200 → valor: 7
    izquierdo: null
    derecho: null

0xC600 → valor: 20
    izquierdo: null
    derecho: null
```

Otra vez:
> **Todos son direcciones de Memoria**.

### La Regla Universal
Casi todas las estructuras complejas se reducen a esto:
> dato + referencia

Ejemplo General:
```

estructura {
    datos
    punteros
}
```

Esto construye:
- [[05. Arreglos|Listas]]
- [[11. Árboles|Árboles]]
- [[09. Grafos|Grafos]]
- [[10. Tablas Hash|Tablas HASH]]
- [[12. Árboles → Heaps|Heaps]]
- Etc.

Todo nace de combinar datos + Direcciones de Memoria.

### Ejercicio

Tenemos esto:
```python

a = [1, 2]
b = [3, 4]

c = [a, b]
```

>**¿Cómo se vería esto en Memoria?**

Pues, se vería así:
```

STACK
a → 0xA100
b → 0xA200

c → 0xC100

HEAP
0xA100 = Lista (Array)
    |
    |---0xB100 = 1(00000001)
    |---0xB200 = 2(00000010)

0xA200 = Lista (Array)
    |
    |---0xB600 = 3(00000011)
    |---0xB700 = 4(00000100)

0xC100 = Lista (Array)
    |
    |---0xA100 = Lista (Array)
    |---0xA200 = Lista (Array)
```
