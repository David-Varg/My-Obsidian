🔡 Representación de Texto y Símbolos (Encoding)
 - ¿Cómo sabe la CPU que 01000001 es una A y no el número 65? Por el Contexto (MIME Types / Headers).
	- **ASCII**: Estándar de 7-8 bits.
	- 010 al inicio = MAYÚSCULA.
	- 011 al inicio = minúscula.
	- Los últimos 5 bits = Posición en el abecedario (A=1, B=2... X=26).
- **UTF-8 (Unicode)**: El estándar de la Web.
	- Usa Leading Bits para saber cuántos bytes leer.
	- Si empieza con 11110..., la CPU sabe que debe unir los siguientes 3 bytes para formar un Emoji: 😈.
